基本型

TypeScript で使われる基本的な型について解説します。以下は、TypeScript でよく使われる型のいくつかです。 

let isDone: boolean = true;
let age: number = 25;
let username: string = "John Doe";

上記のコードでは、`boolean` 型、`number` 型、`string` 型を使っています。これらは TypeScript での基本的な型です。
