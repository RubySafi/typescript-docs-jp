style 属性

HTML では，各タグは style という属性をもつ．style 属性を設定すると，そのタグ内に対し，

  • 文字の大きさを変更する
  • 文字に色を付ける
  • 背景色を設定する

など，様々な装飾を付与することが可能になる．style の付与の仕方は，style 属性を利用する以外にも色々な方法があるが， ここでは入門的な方法として，style 属性を直接使う方法を紹介する．また，すべての style を紹介するのは分量的に困難なため， DOM 操作において使いうる，有用で分かりやすいもののみ紹介することにとどめておく．

style は，style = "color: red;"のように，プロパティ名(=key)とその値(=value)の組み合わせで設定し，最後にセミコロンを付与しなければならない． 複数の style を同時に設定する場合は，style = "color: red; background-color:white;"のように複数並べるだけでよい．

リスト

複数の項目を箇条書き式に並べることを実現させるときに用いるのがリストである．主に使われるリストは，以下の2種類である：

  • 番号無しリスト（unordered list）．列挙することを目的とし，項目の順は無関係である．
  • 順序付きリスト（ordered list）．既定の順に列挙することを目的とし，項目の順が重要である．

Unordered List

「番号無しリスト」環境を作るには，まずは<ul>タグで囲む必要がある．そのタグ領域内に，<li>タグで，表示したい項目を囲む．

  • 項目1
  • 項目2
  • 項目3
<ul>
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ul>

Ordered List

「順序付きリスト」環境を作るには，まずは<ol>タグで囲む必要がある．そのタグ領域内に，<li>タグで，表示したい項目を囲む．

  1. 項目1
  2. 項目2
  3. 項目3
<ol>
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ol>

スタイル適用例

style を適用することで，表示を切り替えることができる．

  1. 項目1
  2. 項目2
  3. 項目3
<ol style="list-style-type: circle;">
  <li>項目1</li>
  <li style="color:green; font-size:20px;">項目2</li>
  <li>項目3</li>
</ol>

上記では，<ol>タグにスタイル「list-style-type: circle;」を適用したため，内部のリスト項目の記号が白丸に変化した． 一方，2つめの<li>タグにのみスタイル[color:green; font-size:20px;」を適用したため，二つ目のリストのみ色が緑色に，フォントサイズが20pxに変化した．

スタイルを設定する際，各スタイルの区切りである「セミコロン」は忘れやすいので注意．

tableタグ

リストは一次元的に項目を配置するのに対し，table は二次元的に項目を配置する．そのような表を実現させるときに使用するタグが <table>タグである．table タグに関係する基本的なタグは以下である．

タグ 説明
<table>～</table> tableタグ．囲まれた部分が table 環境になる．
<tr>～</tr> table row タグ．table において，行環境を作り， この行環境が一つの行を構成する．
<th>～</th> table header タグ．行環境において，列ヘッダーセルを表示する環境を作る．
<td>～</td> table data タグ．行環境において，データセルを表示する環境を作る．

例

名前 年齢
Alice 23
Bob 18 
<table>
  <tr><th>名前</th><th>年齢</th></tr>
  <tr><td>Alice</td><td>23</td></tr>
  <tr><td>Bob</td><td>18</td></tr>
</table>

上記の例のように，一つ一つの行は，tr タグで囲むことにより実現されている． また，headerセルである「名前」「年齢」は th タグで囲んでいる．各セルは，td タグで囲まれている．

スタイル適用例

名前 年齢
Alice 23
Bob 18 
<table style="border-color:red; border-width: 2px; border-style:solid;">
  <tr><th style="text-align: center;">名前</th><th>年齢</th></tr>
  <tr><td>Alice</td><td style="border-style:dashed; border-width:2px;">23</td></tr>
  <tr><td>Bob</td><td>18</td></tr>
</table>

table等に罫線の設定をする際，border-style, border-width, border-color を設定しないといけない点に注意． これらは border キーを使えば， border: 1px solid black;のように一括で登録することもできて便利である

すべてのセルに対して個別に設定をするのは面倒かもしれないが，スタイルシート（css）を利用したり，プログラム側で設定をすることができるが，これは後のページで紹介する．
